HTML Structure

Luxen is bootstrap 3 based responsive template. You can use all of bootstrap features. More information Bootstrap3.

CSS Structure

Master css file is main.css. Other css files are imported it. You can modify all of css files just be careful.

CSS Files:

1- main.css (Main css file)

2- bootstrap-min.css (Main bootstrap css files)

3- font-awesome.min.css (FontAwesome css file)

4- prettyPhoto (prettyPhoto Css)

5- 2035-reset.css (Reset css for luxen)

6- 2035.responsive.css (CSS for luxen responsive features)

7- flexslider.css

Javascript

All of js files in js folder. Master and custom js file is main.js. You can edit it easily. You can find comment lines in main.js file.

Javascript Files:

1- bootstrap.min.js (Main bootstrap 3 js file)

2- respond.min.js (ie8 responsive js file)

3- jquery.fitvids.js (Fitvids js file)

4- jquery.prettyPhoto.js (prettyPhoto js file)

5- gmaps.js (Simple google map usage js file)

6- supersized.js (supersized js file)

7- supersized.shutter.min.js(supersized shutter js file)

8- main.js (Other library triggers and special functions for theme)

And other js files are main funtions library js files.

Customization

Luxen has easy customization features.

COLOR

You can change color from main.css Here is the main colors; First : #0c242e Second : #e4b248.

FONT

We are using google web fonts you can watch this video how can you change fonts.

GOOGLE MAPS

Open main.js file and change line 147/148. You can use here for find latitude and longitude http://itouchmap.com/latlong.html

RESERVATION FORM

Open reservation.php and change the mail address from line 30.

CONTACT FORM

Open mail.php and change the mail address from line 3.

MAILCHIMP INTEGRATION

- Take your newsletter form from mailchimp. Documentation

- Open index-mailchimp.html and go to line 480 and change this url http://2035themes.us7.list-manage.com/subscribe/post?u=3e0321e1bdae39396f013fe32&id=e7043d3e23 with yours.

- And change all footer newsletter form with mailchimp form.

WEATHER

Open main.js file and change line 165. It's using Yahoo Weather
