Layer Structure

All psd file is fully layered. Here is the structure of layers and groups.
Here is the basic folder structure.



Header = All header items are included in this folder.
Slider = Slider elements are here.
Content = Here is the customizable area. This folder items are changing with page type.
Footer = All footer elemnts are in this folder.

Here is the page layer structure.



Each page items are located in content folder. By the way each sections are fully layered and foldered. You can find all items from layer name.
